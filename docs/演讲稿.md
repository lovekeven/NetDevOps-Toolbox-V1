**（展示拓扑发现代码 - 文件：core/topology/snmp_collector.py）**

这是我们的拓扑发现核心代码。通过SNMP协议采集设备信息，使用LLDP协议发现邻居关系。

```python
# 核心代码展示
async def snmp_walk(self, oid):
    """SNMP WALK 获取设备信息"""
    results = []
    current_oid = ObjectType(ObjectIdentity(oid))
    while True:
        error_indication, error_status, error_index, var_binds = await next_cmd(
            self.snmp_engine, self.auth_data, self.transport_target,
            ContextData(), current_oid, lexicographicMode=False
        )
        if error_indication or error_status or not var_binds:
            break
        for var_bind in var_binds:
            results.append((str(var_bind[0]), var_bind[1]))
    return results
```

```javascript
// 终端输入处理
terminal.onData(data => {
    if (data === '\r') {
        // 回车执行命令
        executeCommand(commandBuffer.trim());
    } else if (data === '\x7f') {
        // 退格删除
        commandBuffer = commandBuffer.slice(0, -1);
    } else {
        // 本地回显
        commandBuffer += data;
        terminal.write(data);
    }
});
```
